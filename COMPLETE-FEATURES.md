### Game 1: Tap Hearts 💕
- Tap 10 floating hearts
- Each shows romantic Instagram-worthy message
- Hearts float with animation
- Disappear after tapping
- Counter shows progress

- Unscramble romantic words (BEAUTIFUL, FOREVER, PRECIOUS, BIRTHDAY)
- Tap letters to build word
- Reset button if wrong
- Shake animation on error
- Instant success on correct

### Game 3: Love Quiz 💘 ⭐ WITH PHOTO POPUPS!
- 5 relationship questions with obvious answers
- **Photo popup after each correct answer!**
- **Romantic caption with each photo**
- **Bouncy animation**
- **3-second display**
- **5 dummy quiz photos included (quiz1-5.svg)**
- Green flash on correct
- Shake on wrong
- Progress counter (Question X/5)

### Game 4: Memory Match 🎴
- Match 3 pairs of photo cards
- Tap to flip cards
- Cards stay flipped on match
- Flip back if no match
- Match counter
- Uses photo1-3.svg

### Game 5: Timeline Sort 📅
- **Tap-to-swap system** (fixed drag-drop issue!)
- 4 photos to arrange chronologically
- **Number badges (1, 2, 3, 4)**
- **Yellow border on selected**
- **Pink border normally**
- Tap photo 1 → tap photo 2 → they swap!
- Check Order button
- Uses photo1-4.svg

### Game 6: Complete the Line ✨
- Fill in missing word in romantic phrases
- 4 multiple choice options
- Examples: "In a room full of art, I'd still stare at ____"
- Shake on wrong answer
- Instant success on correct

### Game 7: Sliding Puzzle 🧩 ⭐ SUPER ENHANCED!
- 3x3 tile puzzle with photo
- **Number badges (1-9) on each tile**
- **Directional hints!**
  - **Green border** on moveable tiles
  - **Arrow emoji** (⬆️ ⬇️ ⬅️ ➡️)
  - **Text label** ("Slide Up", etc.)
  - **Pulsing animation**
- **Shuffle counter** tracks attempts
- **Skip button appears after 5 shuffles!**
  - Green button with pulse
  - "Skip Puzzle ⏭️"
- Shuffle Again button
- Uses puzzle.svg

### Game 8: Final Reveal 🎁
- Clear, unblurred gift photo
- **Your heartfelt message** (customize in HTML)
- Beautiful card layout
- "I Love You Too!" button
- Leads to celebration screen
- Uses gift.svg
