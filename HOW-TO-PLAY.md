# 🎮 How to Play Each Game

## Game 1: Tap Hearts 💕
- **Goal:** Tap all 10 floating hearts
- **How:** Simply tap/click each heart
- **Result:** Each heart shows a romantic message
- **Complete:** Tap all 10 hearts

---

## Game 2: Word Scramble 🔤
- **Goal:** Unscramble the romantic word
- **How:** Tap letters in the correct order
- **Words:** BEAUTIFUL, FOREVER, PRECIOUS, BIRTHDAY
- **Reset:** Use the Reset button if you make a mistake
- **Complete:** Spell the word correctly

---

## Game 3: Love Quiz 💘
- **Goal:** Answer 5 relationship questions
- **How:** Tap the correct answer
- **Hint:** The answers are obvious! (It's always "Me" or "You")
- **Example:** "Who loves you more?" → Answer: "Me, forever"
- **Complete:** Answer all 5 questions correctly

---

## Game 4: Memory Match 🎴
- **Goal:** Match 3 pairs of photo cards
- **How:** Tap two cards to flip them
- **Match:** If they match, they stay flipped
- **No Match:** If they don't match, they flip back
- **Complete:** Match all 3 pairs

---

## Game 5: Timeline Sort 📅 ⭐ NEW CONTROLS!
- **Goal:** Arrange memories from oldest to newest
- **How to Swap:**
  1. **Tap the FIRST photo** you want to swap → It gets a yellow border
  2. **Tap the SECOND photo** → They swap positions!
  3. Keep swapping until the order is correct
- **Cancel:** Tap the same photo again to deselect
- **Numbers:** Each photo has a number (1, 2, 3, 4) showing current position
- **Check:** Press "Check Order" button when you think it's correct
- **Complete:** Get all 4 memories in chronological order

**Example Timeline:**
```
Position 1: First Meet
Position 2: First Date
Position 3: Our Anniversary
Position 4: Recent Memory
```

---

## Game 6: Complete the Line ✨
- **Goal:** Fill in the missing word in a romantic phrase
- **How:** Choose the correct word from 4 options
- **Example:** "In a room full of art, I'd still stare at ____"
  - Options: "the paintings", "you", "the ceiling", "my phone"
  - Correct: "you"
- **Complete:** Choose the right word

---

## Game 7: Sliding Puzzle 🧩
- **Goal:** Arrange the tiles to complete the photo
- **How:** Only tiles next to the empty space can move - look for GREEN borders!
- **Visual Hints:**
  - **GREEN BORDER** = This tile can move!
  - **Arrow + Text** = Shows direction (⬆️ Slide Up, ⬇️ Slide Down, ⬅️ Slide Left, ➡️ Slide Right)
  - **Pink border** = This tile can't move right now
- **Numbers:** Each tile has a number (1-9) showing its correct position
- **Strategy:**
  - Only click tiles with green borders
  - Follow the arrow hints to know which direction they'll slide
  - Work on corners first, then edges
- **Shuffle:** Use "Shuffle Again" if you want to restart with a new puzzle
- **Shuffle Counter:** Shows how many times you've shuffled
- **Skip Option:** After shuffling 5 times, a green "Skip Puzzle ⏭️" button appears!
  - Too hard? Just shuffle 5 times and skip it!
- **Complete:** Get all tiles in the correct position OR skip after 5 shuffles
- **Difficulty:** With the visual hints, it's much easier! But you can still skip if needed 😊

## 🐛 Troubleshooting

**Images not showing?**
- Dummy placeholder images will show colored boxes with "Replace Me"
- This is normal until real photos are added

**Timeline not swapping?**
- Make sure to tap TWO different photos
- First tap = select (yellow border)
- Second tap = swap with first
- Tap same photo = cancel selection

**Game stuck?**
- Refresh the page (you'll start from Game 1)
- Make sure you're on a modern browser (Chrome, Safari, Firefox)
