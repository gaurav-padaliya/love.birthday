The sliding puzzle (Game 7) now shows you EXACTLY which tiles can move

🎯 HOW IT WORKS:
--------------

Only tiles NEXT TO the empty space can move.
Those tiles now show:

✅ GREEN BORDER (instead of pink)
✅ GREEN GLOW/SHADOW around the tile
✅ ARROW EMOJI showing direction (⬆️ ⬇️ ⬅️ ➡️)
✅ TEXT saying "Slide Up", "Slide Left", etc.


📱 VISUAL DESIGN:
---------------

Can Move Tiles:
- Green border (3px thick)
- Green glowing shadow
- Pulsing arrow overlay in center
- Text label below arrow
- Hover effect (scales up slightly)

Can't Move Tiles:
- Normal pink border
- No hints
- Can't click

Empty Space:
- Dashed pink border
- Light pink background
- No hints

🎮 HOW TO PLAY NOW:
-----------------

1. Look for tiles with GREEN BORDERS
2. Read the arrow hint (⬆️ Slide Up, etc.)
3. Click that tile
4. It slides in the shown direction!
5. Hints update after each move

-------------

State 1 - Can Move:
→ Green border + glow

→ Clickable, scales on hover

State 2 - Can't Move:

State 3 - Empty Space:

→ No hints, not clickable
